# Stats Dashboard

This is a simple analytics dashboard web page built with HTML, CSS, and JavaScript.

## Files

- `index.html` — page structure and content
- `style.css` — dashboard styling and responsive layout
- `script.js` — live data simulation, clock, controls, and activity log

## Responsive behavior

The dashboard now supports multiple screen sizes:

- Desktop: wide layout with shadows and spacing
- Tablet: grid adjusts to smaller cards and header elements stack
- Mobile: controls become full-width, card layout becomes compact, text scales down

## How to run

1. Open `index.html` in your browser.
2. The dashboard will start updating automatically.
3. Use the buttons:
   - `Auto update` / `Stop auto` — start or stop live simulation
   - `+ Simulate event` — manually change all stats
   - `↺ Reset` — restore original values and clear the log

## Notes

- The page includes a live clock and activity log.
- It uses CSS grid and media queries for responsive layout.
- The dashboard is designed to look good on phones, tablets, and desktops.
